# 性能

作为 NVIDIA NeMo 框架的一部分，Megatron Bridge 通过整合最新的训练技术（如模型并行化、优化的注意力机制等），为训练先进的生成式 AI 模型提供最佳性能，以实现高训练吞吐量。

本页提供了在不同 GPU 系统和配置下使用 Megatron-Bridge 的大型语言模型的性能基准数据。

## 术语表

- **GBS**: 全局批次大小
- **MBS**: 微批次大小
- **FSDP**: 全分片数据并行
  - FSDP > 0: 使用 FSDP，分片组大小 = #GPUs / (TP × PP)
  - FSDP = 0: 使用 DDP（分布式数据并行）
- **TP**: 张量并行大小
- **PP**: 管道并行大小
- **CP**: 上下文并行大小
- **VP**: 虚拟管道并行大小
- **EP**: 专家并行大小
- **GA**: 梯度累积次数

## 性能指标

性能使用以下指标衡量：

- **Tokens/sec/GPU**: 每 GPU 吞吐量
- **Model TFLOP/sec/GPU**: 每 GPU 模型浮点运算次数每秒

## 大型语言模型性能摘要

以下是按发布版本组织的各种大型语言模型的性能基准数据。这些结果是使用[此处](https://github.com/NVIDIA-NeMo/Megatron-Bridge/tree/main/scripts/performance)提供的性能配方获得的。

性能数据包括：

- **预训练性能**: 各种模型大小和架构的吞吐量指标
- **系统配置**: 在不同 GPU 系统（DGX-GB300、DGX-GB200、DGX-B300、DGX-B200、DGX-H100）上的结果
- **精度选项**: 不同精度模式（BF16、FP8、MXFP8）之间的性能比较
