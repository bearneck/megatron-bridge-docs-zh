# 优化器和调度器配置

优化器和调度器配置控制优化算法、学习率调度和权重衰减策略。

## OptimizerConfig（来自 Megatron Core）

`OptimizerConfig` 包含优化算法的所有参数，直接来自 Megatron Core。关键参数包括：

| 参数 | 类型 | 描述 |
|-----------|------|-------------|
| `optimizer` | `str` | 优化器类型（"adam"、"sgd" 等） |
| `lr` | `float` | 基础学习率 |
| `min_lr` | `float` | 衰减调度中的最小学习率 |
| `weight_decay` | `float` | L2 正则化系数 |
| `adam_beta1` | `float` | Adam 优化器 beta1 参数 |
| `adam_beta2` | `float` | Adam 优化器 beta2 参数 |
| `adam_eps` | `float` | Adam 优化器 epsilon 参数 |
| `clip_grad` | `float` | 梯度裁剪阈值 |
| `use_distributed_optimizer` | `bool` | 启用分布式优化器以提高内存效率 |
| `overlap_grad_reduce` | `bool` | 梯度减少与计算重叠 |
| `overlap_param_gather` | `bool` | 参数收集与计算重叠 |
| `bf16` | `bool` | 使用 BF16 精度进行训练 |
| `fp16` | `bool` | 使用 FP16 精度进行训练 |

## SchedulerConfig

`SchedulerConfig` 控制整个训练过程中的学习率调度和权重衰减进程。

### 学习率调度

| 参数 | 类型 | 默认值 | 描述 |
|-----------|------|---------|-------------|
| `lr_decay_style` | `Literal["constant", "linear", "cosine", "inverse-square-root", "WSD"]` | `"linear"` | 学习率衰减函数 |
| `lr_decay_iters` | `Optional[int]` | `None` | 学习率衰减所持续的迭代次数（默认为 `train_iters`）。用于基于迭代的训练。 |
| `lr_decay_samples` | `Optional[int]` | `None` | 学习率衰减所持续的样本数（默认为 `train_samples`）。用于基于样本的训练。 |
| `lr_warmup_iters` | `int` | `0` | 线性预热学习率的迭代次数。用于基于迭代的训练。 |
| `lr_warmup_samples` | `int` | `0` | 线性预热学习率的样本数。用于基于样本的训练。 |
| `lr_warmup_fraction` | `Optional[float]` | `None` | 用于预热的衰减迭代次数/样本数的比例（两种模式都适用） |
| `lr_warmup_init` | `float` | `0.0` | 预热阶段的初始学习率 |

**调度器模式选择**

调度器支持两种模式，必须与您的训练配置保持一致：

1.  **基于迭代的调度**：将 `lr_decay_iters` 和 `lr_warmup_iters` 与 `train_iters` 配合使用。
2.  **基于样本的调度**：将 `lr_decay_samples` 和 `lr_warmup_samples` 与 `train_samples` 配合使用。

**重要约束**
- 不能混合使用基于迭代和基于样本的调度器参数。
- 您的调度器模式必须与您的训练模式（基于迭代 vs 基于样本）匹配。
- `lr_warmup_fraction` 与两种模式都兼容，但不能与显式指定的预热迭代次数/样本数同时使用。

### WSD（预热-稳定-衰减）调度

| 参数 | 类型 | 默认值 | 描述 |
|-----------|------|---------|-------------|
| `lr_wsd_decay_style` | `Literal["exponential", "linear", "cosine"]` | `"exponential"` | WSD 衰减阶段的衰减风格 |
| `lr_wsd_decay_iters` | `Optional[int]` | `None` | WSD 衰减阶段的迭代次数。用于基于迭代的训练。 |
| `lr_wsd_decay_samples` | `Optional[int]` | `None` | WSD 衰减阶段的样本数。用于基于样本的训练。 |

### 权重衰减调度

用于控制训练期间权重衰减进程的参数，包括起始值和结束值以及调度策略：

| 参数 | 类型 | 默认值 | 描述 |
|-----------|------|---------|-------------|
| `start_weight_decay` | `Optional[float]` | `None` | 初始权重衰减系数 |
| `end_weight_decay` | `Optional[float]` | `None` | 最终权重衰减系数 |
| `weight_decay_incr_style` | `Literal["constant", "linear", "cosine"]` | `"constant"` | 权重衰减进程风格 |

### 检查点集成

用于管理在加载检查点时如何应用调度器设置的参数，允许控制是优先使用配置值还是从保存的状态恢复：

| 参数 | 类型 | 默认值 | 描述 |
|-----------|------|---------|-------------|
| `override_opt_param_scheduler` | `bool` | `False` | 从配置重置调度器值，忽略检查点 |
| `use_checkpoint_opt_param_scheduler` | `bool` | `False` | 使用检查点中的调度器值，忽略配置 |

### 计算字段

这些字段在配置验证期间自动计算，有助于将训练计划与配置的批次大小和迭代次数对齐：

| 字段 | 描述 |
|-------|-------------|
| `lr_warmup_steps` | 预热的总步数（根据迭代次数和批次大小计算） |
| `lr_decay_steps` | 衰减的总步数（根据迭代次数和批次大小计算） |

| `wd_incr_steps` | 权重衰减递增的总步数 |
| `wsd_decay_steps` | WSD 退火阶段的总步数 |

## 学习率调度策略

以下调度策略定义了训练期间学习率如何变化，每种策略适用于不同的收敛行为和模型类型：
| 调度类型           | 描述                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| **恒定**            | 学习率在整个训练期间保持固定。                            |
| **线性**              | 学习率从基础学习率线性下降到最小学习率。        |
| **余弦**              | 学习率遵循从基础学习率到最小学习率的余弦衰减曲线。      |
| **反平方根** | 学习率按步数的反平方根比例衰减。 |

## WSD（预热-稳定-衰减）
WSD 调度将学习率变化过程划分为三个不同的阶段，为早期上升、中期稳定和最终衰减提供细粒度控制：
| 阶段     | 描述                                              |
|-----------|----------------------------------------------------------|
| **预热** | 学习率从初始值线性增加到基础学习率。 |
| **稳定** | 学习率在基础学习率处保持恒定。              |
| **衰减**  | 学习率使用指定样式（例如，指数、线性、余弦）衰减到最小学习率。 |

## 权重衰减调度

这些调度选项控制权重衰减系数如何随时间变化，允许适应不同训练阶段的正则化策略：
| 调度类型 | 描述                                                                 |
|---------------|-----------------------------------------------------------------------------|
| **恒定**  | 整个训练期间使用固定的权重衰减。                                     |
| **线性**    | 从起始权重衰减值线性变化到结束值。                          |
| **余弦**    | 从起始权重衰减值余弦变化到结束值。                          |
